# Python Visualization – B2B Dataset

This project is part of my internship portfolio.  
It demonstrates how to use **Python for data preprocessing and visualization** with libraries like `pandas`, `matplotlib`, and `seaborn`.

## 📊 Features
- Data preprocessing with Pandas
- API data extraction (Onzitr, Alpha Vantage)
- Exploratory Data Analysis (EDA)
- Visualizations:
  - Monthly revenue trends (line plot)
  - Customer segment analysis (histogram)
  - Product category distribution (pie chart)
  - Correlation heatmap

## 🛠️ Tech Stack
- Python (Pandas, Matplotlib, Seaborn, Requests)
- Jupyter Notebook
- REST API Integration
